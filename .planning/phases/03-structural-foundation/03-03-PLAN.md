public class NoticiasBusiness : INoticias

        // OR for declaration syntax: using var cx = _factory.Create();
    }
}

1. Add `private readonly IDataContextFactory _factory;` field
2. Add `IDataContextFactory factory` parameter to constructor
3. Add `_factory = factory;` in constructor body
4. Replace every `new PORTALNEGOCIODataContext()` and `new()` (when typed as PORTALNEGOCIODataContext) with `_factory.Create()`
5. Keep `using` keyword — just change the expression
6. Do NOT change any business logic, LINQ queries, or method signatures
